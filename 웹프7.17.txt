target의 속성값의 종류 
1. _blank: 새로운 웹 브라우저 창에 연다 
2. _self: 현재 웹 브라우저 창에 연다 
3. _parent: 부모 웹 브라우저 창에 연다 
4. _top: 웹 브라우저 전체 영역에 연다 
- 문저 내 특정 위치로 이동 
id 속성값을 쓸 때에는 중복값이 없어야 함. 하지만 name은 중복값이 허용됨. 그룹으로 속성을 지정할 수 있음.
